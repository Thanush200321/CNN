# Fashion MNIST CNN From Scratch

## Objective
Build a CNN from scratch using NumPy and train it on Fashion MNIST dataset.

## Features
- Convolution Layer
- Max Pooling Layer
- Flatten Layer
- Fully Connected Layer
- Forward Pass
- Backward Pass
- CNN Training

## Technologies Used
- Python
- NumPy
- Matplotlib
- Google Colab

## Dataset
Fashion MNIST Dataset

## Accuracy
Achieved around 80%+ accuracy.

## How to Run
1. Open notebook in Google Colab
2. Run all cells sequentially
